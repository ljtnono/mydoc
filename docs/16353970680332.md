# Homebrew常用命令


* 安装任意包

```bash
$ brew install <packageName>
```

示例：安装node

```bash
$ brew install node
```

* 卸载任意包

```bash
$ brew uninstall <packageName>
```

示例：卸载git

```bash
$ brew uninstall git
```

* 查询可用包

```bash
$ brew search <packageName>
```

* 查看已安装包列表

```bash
$ brew list
```

* 查看任意包信息

```bash
$ brew info <packageName>
```

* 更新Homebrew

```bash
$ brew update
```

* 查看Homebrew版本

```bash
$ brew -v
```

* Homebrew帮助信息

```bash
$ brew -h
```
